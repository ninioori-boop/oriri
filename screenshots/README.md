# צילומי מסך לסקשן "סיור במערכת" (advisors.html)

שמרו כאן את צילומי המסך של המערכת בשמות הבאים בדיוק.
הסקשן בדף יציג אותם אוטומטית במקום ה-placeholders.

| שם הקובץ                     | מה אמור להיות בצילום                                  |
|------------------------------|------------------------------------------------------|
| `mapping-categories.png`     | טאב מיפוי — הוצאות לפי קטגוריה (ביגוד, אוכל, ביטוח...) |
| `cashflow.png`               | תזרים חודשי נטו — עדכון חי (הכנסות מול הוצאות, נטו)    |
| `monthly-budget.png`         | תקציב חודשי — תכנון מול ביצוע (תקציב ינואר)           |
| `trends.png`                 | מגמות — ניתוח שנתי + גרף הכנסות מול הוצאות            |
| `expense-log.png`            | תיעוד הוצאות — יומן הוצאות                            |
| `checking.png`               | התנהלות עו"ש — עודף חודשי                             |
| `goals.png`                  | יעדים פיננסיים                                        |
| `loans.png`                  | מחשבון הלוואות — לוח סילוקין                          |

## טיפים
- אפשר גם `.jpg`/`.webp`, אבל אז עדכנו את הסיומת ב-`advisors.html` (חפשו `screenshots/`).
- כדאי לחתוך את צילומי המסך נקי (בלי שורת הכתובת/הדפדפן).
- רוחב מומלץ: 1200px ומעלה, כדי שיהיו חדים.
- אחרי שתשמרו את כולם, הסקשן ייראה מלא וניתן לדחוף לאתר.
